# SeeHydro 训练手册

这份文档只解决一件事：你自己准备好切片和标签之后，训练怎么开始。

## 1. 先说实话

当前仓库不是“一条命令自动训练完”的成熟产品。

更准确地说：

- 训练模块已经有了
- 训练逻辑已经写了一部分
- 但你还需要自己整理数据、决定先做什么任务

## 2. 新手先练什么

最推荐第一阶段只练：

- 分割任务
- 只识别背景和水面

不要第一步就同时做：

- 检测任务
- 多类别结构物识别
- 大规模全量数据训练

## 3. 分割训练前你要准备什么

至少准备：

- 图像切片目录
- 掩膜目录
- 图像和掩膜同名

参考结构：

```bash
data/train_seg/images
data/train_seg/masks
```

例如：

```bash
data/train_seg/images/tile_001.tif
data/train_seg/masks/tile_001.tif
```

## 4. 仓库里的分割训练入口

关键代码在：

- [`train_seg.py`](/tmp/SeeHydro/src/seehydro/training/train_seg.py)
- [`dataset.py`](/tmp/SeeHydro/src/seehydro/training/dataset.py)

代码层面的真实要求包括：

- 默认会自动把数据拆成训练集和验证集
- 验证集比例约 20%
- 默认模型是 `DeepLabV3Plus`
- 默认编码器是 `resnet101`
- 默认训练轮数偏大，第一次不建议直接用满

## 5. 训练时你要注意什么

第一次训练建议保守一点：

- 先用小数据集
- 先用较少 epoch
- 先看训练能不能跑完

你的第一目标不是高精度，而是：

- 代码能跑
- 模型能收敛
- 推理能出结果

## 6. 检测训练怎么理解

如果你后面做桥梁、倒虹吸、渡槽等结构物检测，要走检测训练路线。

关键代码在：

- [`dataset.py`](/tmp/SeeHydro/src/seehydro/training/dataset.py)
- [`train_det.py`](/tmp/SeeHydro/src/seehydro/training/train_det.py)

但第一阶段不建议先走这条线，因为它对标签格式要求更严格，也更容易卡住。

## 7. 推荐训练顺序

最推荐的顺序是：

1. 下载一小块影像
2. 切片
3. 做少量分割标注
4. 先跑分割训练
5. 再考虑检测训练

## 8. 训练阶段最容易踩的坑

- 图像和掩膜文件名对不上
- 掩膜不是类别 ID 图
- 一开始就训练太大数据集
- 一开始就设很复杂类别
- 还没验证最小闭环，就追求高精度

## 9. 一句话结论

当前仓库最合理的第一条训练路线是：

```text
天地图/本地影像 -> 切片 -> 背景/水面分割标注 -> 分割训练
```
