# SeeHydro 标注与数据集手册

这份文档只解决一件事：下载和切片之后，你自己的数据应该怎么整理，后面训练才不会卡住。

## 1. 先说结论

对于当前仓库：

- 下载后不会自动生成训练标签
- 标注通常还是要你自己做
- 新手建议先做分割，不要先做检测
- 最推荐的最小路线是：只做“背景 + 水面”分割

## 2. 推荐你先做什么

建议第一阶段只做最小闭环：

1. 下载一小块影像
2. 切片
3. 只标 30 到 100 张
4. 先做分割
5. 先只标最关键类别

不要一开始就同时做：

- 分割 + 检测
- 全线大范围数据
- 很多类别同时标

## 3. 分割数据集格式

当前仓库里的分割数据集类要求：

- 图像目录：`images/`
- 掩膜目录：`masks/`
- 图像切片：`.tif`
- 掩膜文件：`.tif`
- 图像和掩膜文件名必须同名

对应代码在：
- [`dataset.py`](/tmp/SeeHydro/src/seehydro/training/dataset.py)

推荐目录结构：

```bash
data/train_seg/images
data/train_seg/masks
```

例如：

```bash
data/train_seg/images/tile_001.tif
data/train_seg/masks/tile_001.tif

data/train_seg/images/tile_002.tif
data/train_seg/masks/tile_002.tif
```

重点：

- 文件名要一一对应
- 掩膜必须是单通道
- 掩膜像素值必须是类别 ID，不是普通彩色图

## 4. 分割类别建议

仓库默认的分割类别设定是：

- `0` 背景
- `1` 渠道水面
- `2` 渠道边坡
- `3` 马道
- `4` 管理道路

但如果你只是先跑通，建议你先做最简单版本：

- `0` 背景
- `1` 水面

先不要一开始就标满 5 类。

## 5. 掩膜图要求

训练用的掩膜图不是随便画出来的彩色图，而是：

- 单通道
- 每个像素值表示一个类别编号

例如：

- 背景像素值写成 `0`
- 水面像素值写成 `1`

如果标签值乱了，训练就会直接出问题。

## 6. 标注工具怎么选

新手建议选一个你能稳定用的工具，不要太纠结。

常见选择：

- Labelme
- CVAT
- Label Studio

如果你只是想先起步，优先选轻量一些的工具。

## 7. 检测数据集格式

如果你后面要做桥梁、倒虹吸、渡槽等检测，当前仓库代码按 YOLO 数据格式理解。

推荐结构：

```bash
data/train_det/
  images/
    train/
    val/
  labels/
    train/
    val/
  data.yaml
```

标签文件是 `.txt`，每行通常是：

```text
class_id x_center y_center width height
```

当前仓库相关代码也在：
- [`dataset.py`](/tmp/SeeHydro/src/seehydro/training/dataset.py)

## 8. 为什么建议你先做分割

因为当前项目里，分割更适合你先做“水面识别”这条最小路线。

原因：

- 水面区域连续，更容易先看到结果
- 数据组织相对直接
- 对新手更容易验证模型有没有学到东西

## 9. 最小可行方案

如果你现在自己继续做，建议按这个最小方案：

1. 用天地图下载一小块影像
2. 切片
3. 从切片中挑 30 到 100 张
4. 只做背景/水面分割标注
5. 先训练一个最小模型

## 10. 一句话结论

下载只是第一步。

你后面真正要自己准备的是：

- 切片后的训练图像
- 对应标签
- 明确类别定义

最推荐的新手路线：

```text
天地图下载 -> 切片 -> 背景/水面标注 -> 分割训练
```
